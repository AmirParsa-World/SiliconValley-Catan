باران کل منطق بازی تا همین لحظه تموم شده، باید فقط گرافیکش رو بزنیم.
خودم گیم انجین رو هم زدم فعلا.
بعدا اگه کمبودی بود میایم سراغش.
یه مین تستی هم درست کردم، رفع باگ کردم همه چیشو، متود مین هم فصله، تقریبا همه نکات داک بازی چک شده توش.
فقط برای گرافیک بنظر کارمون سخت باشه، تلگرامت رو چکککک کن، باید یکمی فیلم ببینیم ک بدونیم چی به چیه، و اینکه بعدش سفت و سخت بچسبیم به بحث گرافیک.
یکسری اکسپشن هم فعلا اضافه کردم راستی،.
برای هماهنگی و تقسیم کار گرافیک هم حتما باید جفتمون فیلم های یکسانی رو ببینیم، تا به یک شکل جلو بریم، و کار هم جمع بشه، تک نفری نمیشه.
